# P1-1 — Hub redesign plan

## Goal

Demote Create/Join from hero-card status, promote OWNED business list to top,
hide search until the user has 4+ businesses, ensure rows feel like list items.

## Approach

1. **JSX reorder in `HubHome.tsx`** (only file touching markup):
   - Greeting (unchanged) → OWNED list → JOINED list → slim mono "+ NEW · JOIN
     WITH CODE" affordance below the lists → empty state (only when zero
     businesses).
   - When there are zero businesses, swap to the previous layout: show the
     two `FeatureCard`s prominently and the empty state — first-run users
     still need the big call to action.

2. **`+ NEW · JOIN WITH CODE` affordance**: render as a slim mono row
   (button) sitting below the OWNED/JOINED list. Tap opens an
   `IonActionSheet` with two buttons:
   - "Create a business" → `openCreateModal()`
   - "Join with a 6-character code" → `openJoinModal()`
   - Cancel
   This reuses the existing Create/Join modal providers — no markup change
   in those modals, just a new entry point.

3. **Search visibility**: gate the `<label className="app-search">` block
   behind `businesses.length >= 4` (constant `SEARCH_MIN_BUSINESSES = 4`).

4. **BusinessRow padding**: already `var(--space-3) var(--space-4)` in
   `app.css` — meets spec. No change.

5. **New CSS (`app.css`)**: add `.hub-compose-row` — a slim full-width
   ghost button with mono caps, terracotta accent for `+ NEW`, separator
   middot, `JOIN WITH CODE` label. Uses tokens only.

6. **i18n**: add keys
   - `hub.compose_new` → "+ NEW"
   - `hub.compose_join` → "JOIN WITH CODE"
   - `hub.compose_aria` → "Add a business"
   - `hub.action_sheet_title` → "Add a business"
   Add real translations in es and ja (the other "registered" locales per
   `packages/shared/src/locales.ts`); for the remaining message files in
   `/messages/`, copy the English string as a placeholder (consistent with
   how non-registered locales are handled today — verify by sampling one
   existing key).

7. **Regenerate** `apps/web/src/i18n/messageIds.d.ts`.

8. **tsc**: `cd apps/web && npx tsc --noEmit`.

## Out of scope

- No changes to Create/Join modals themselves.
- No changes to greeting/date kicker.
- No BusinessRow visual changes.
